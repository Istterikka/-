## 📌 Что это за программа

«Оценщик» — это небольшая настольная программа для статистической оценки качества математических и имитационных моделей. Она помогает ответить на три главных вопроса:

1. **Адекватна ли модель?** (соответствует ли она реальной системе)
2. **Устойчивы ли результаты?** (меняется ли вывод при разной нагрузке)
3. **Чувствительна ли модель?** (какие параметры влияют сильнее всего)

Программа предназначена для студентов, инженеров-исследователей и научных сотрудников, занимающихся компьютерным моделированием.

## 🚀 Быстрый старт (за 1 минуту)

1. **Запустите** `Estimator.exe`
2. **Выберите** вид оценки (Адекватность / Устойчивость / Чувствительность)
3. **Нажмите** «Загрузить данные» и выберите текстовый файл с числами
4. **Введите** уровень значимости (обычно 0,05) и критические значения
5. **Нажмите** кнопку расчёта → получите вердикт
